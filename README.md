# Kenji Hub

エンジニアとしての経験と技術を共有するポートフォリオサイトです。以下の情報で、このプロジェクトの特長や全体像を理解いただけます。

## 🌟 主な特長

- フレームワーク：Next.js 14 (App Router)
- 言語：TypeScript
- レスポンシブデザインであらゆるデバイスに対応
- 静的サイト生成とモダンな設計
- 記事管理システム

## 📂 ページ構成

- **ホームページ** (`/`) - プロジェクト全体の概要を表示
- **プロジェクトページ** (`/projects`) - 開発したプロジェクトの一覧
- **ブログ記事一覧** (`/blog`) - 各種技術記事を掲載
- **記事詳細ページ** (`/blog/[id]`) - 個別のブログ記事を表示

## 🎨 プロジェクト概要

Kenji Hub は、自身のプロジェクトやブログなどを備えたウェブアプリケーションです。以下のディレクトリ構造により、コードベースは整理されています。

```
kenji-hub/
├── src/                 # ソースコード
│   ├── app/            # ページ固有のルーティング
│   ├── components/     # 再利用可能なコンポーネント
│   ├── hooks/          # 共通のカスタムフック
│   └── lib/            # ユーティリティ
├── public/              # 静的ファイル
├── docs/                # ドキュメント
├── .github/             # GitHub設定
└── 設定ファイル
```
