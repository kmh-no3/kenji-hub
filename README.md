# Kenji Hub

エンジニアとしての経験と技術を共有するポートフォリオサイトです。


## 🌐 アクセス

- **サイト**: https://kmh-no3.github.io/kenji-hub/


## 📱 アプリケーション詳細

### ページ構成

- **ホームページ** (`/`) - `/projects` にリダイレクト
- **プロジェクトページ** (`/projects`) - プロジェクト一覧と自己紹介
- **記事一覧** (`/blog`) - ブログ記事一覧
- **記事詳細** (`/blog/[id]`) - 個別記事の詳細ページ

### 機能

- レスポンシブデザイン
- テーマ切り替え（ライト/ダーク/システム）
- ハンバーガーメニュー（モバイル向けフルスクリーンメニュー）
- 目次機能（記事詳細ページ）
- 静的サイト生成
- SNSリンク（X、GitHub、Zenn）
- 記事管理システム

## 🚀 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS, @tailwindcss/typography
- **パッケージ管理**: npm
- **デプロイ**: GitHub Pages / Docker
- **その他**: ESLint, PostCSS, clsx, tailwind-merge
